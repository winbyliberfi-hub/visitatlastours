## 🏔️ Project Overview

Full-featured tourism website for **Atlas Pathway Journeys**, a Marrakech-based tour operator owned and operated by native Imazighen Berber guide **Abdelaziz Aznag**. The site is optimized for SEO, mobile-first, and designed to convert visitors into bookings via WhatsApp, email, and a booking form.

## 🔍 SEO Implementation

### Schema Markup
- **TourOperator** schema on homepage
- **TouristTrip** schema on all individual tour pages
- **FAQPage** schema on faq.html
- **TourOperator + WebSite** in `structured-data.json`

### Target Keywords
- Primary: "Atlas Mountains day trip from Marrakech" (8.1k), "Marrakech day trips" (6.6k), "Atlas Mountains tour" (5.4k)
- Secondary: "Ourika Valley tour", "Agafay desert experience", "Berber village tour", "camel ride Marrakech"

### Technical SEO
- Canonical URLs pointing to `https://visitatlas.tours`
- Open Graph + Twitter Card meta tags on all pages
- Lazy loading on all images
- Alt text on all images
- Semantic HTML5 throughout
- Google Maps embed on all tour pages
- Internal linking structure across all pages
